Common Form command line interface

Usage:
  commonform bookmark BOOKMARK REFERENCE [-l LIB]
  commonform check REFERENCE [-l LIB]
  commonform complete REFERENCE FILE [options]
  commonform fingerprint REFERENCE [-l LIB]
  commonform read REFERENCE [options]
  commonform share REFERENCE [-l LIB]
  commonform summarize REFERENCE [options]
  commonform -h | --help
  commonform --version

Options:
  --color                     Force color terminal output
  -f FORMAT, --format FORMAT  Output format [default: terminal]
  -l LIB, --library LIB       The library to use
